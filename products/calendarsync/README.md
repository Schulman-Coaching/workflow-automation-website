# CalendarSync
## "Smart Scheduling, Zero Hassle"

### Overview
An intelligent scheduling assistant that unifies calendars, automates booking, sends smart reminders, and eliminates no-shows.

### Core Features
- Unified Calendar View
- Smart Booking Pages
- Automated Reminders & Follow-Ups
- Team Scheduling
- Timezone Intelligence

### Target Persona
Service providers, consultants, anyone who lives by their calendar

### Pricing Tier
Freemium model (Free, Pro, Team)

### Files in this directory
- `index.html` - Product landing page
- `features.html` - Detailed features page
- `pricing.html` - Product-specific pricing
- `demo.html` - Interactive demo/tour
- `assets/` - Product-specific images and assets
