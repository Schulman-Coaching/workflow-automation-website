# ChatFlow
## "WhatsApp Automation for Small Business"

### Overview
A WhatsApp Business automation platform that handles customer inquiries, captures leads, sends notifications, and integrates with your other tools.

### Core Features
- Smart Auto-Replies
- Lead Capture & Qualification
- Broadcast & Campaigns
- Order & Appointment Notifications
- Conversation Dashboard

### Target Persona
Small business owner using WhatsApp for customer communication

### Pricing Tier
Tiered by message volume (Starter, Growth, Scale)

### Files in this directory
- `index.html` - Product landing page
- `features.html` - Detailed features page
- `pricing.html` - Product-specific pricing
- `demo.html` - Interactive demo/tour
- `assets/` - Product-specific images and assets
